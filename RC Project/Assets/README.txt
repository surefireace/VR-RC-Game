How To Play:
Left Joystick: Turn/Move RC
Right Joystick: Turn/Aim RC Gun
A: Reset RC orientation
X: Teleport
Y: Toggle Help path
B: Place a marker for RC 
To pick a marker back up, run into it
Right Trigger: Fire RC gun
